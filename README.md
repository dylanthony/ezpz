# EZPZ

CLI tool for automated project intialization.

## Getting Started

### Installation

1. **Clone the repo:**
   ```sh
   git clone https://github.com/admclamb/ezpz.git
   cd ezpz
   ```

2. **Build the CLI tool:**
   ```sh
   go build -o ezpz
   ```

3. (Optional) **Install globally:**
   ```sh
   go install github.com/admclamb/ezpz@latest
   ```

### Running Tests

To run all tests:

```sh
go test ./...
```
